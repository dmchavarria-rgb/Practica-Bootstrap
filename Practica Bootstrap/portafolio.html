Presentación del contenido

Este portafolio forma parte del sitio web desarrollado en clase. Su propósito es organizar los principales elementos aprendidos durante la práctica: estructura HTML, estilos CSS, uso de imágenes, tablas, enlaces internos y adaptación visual para diferentes tamaños de pantalla.

La página mantiene relación con el proyecto original porque resume el proceso de aprendizaje web y utiliza la misma línea visual del sitio.

Bloques de información

HTML semántico

Se utilizaron etiquetas como header, nav, main, section, article, figure y footer para dar una estructura clara al contenido de la página.

CSS y diseño visual

El archivo CSS propio permite personalizar colores, bordes, espaciados, tarjetas y efectos visuales sin modificar la estructura HTML.

Responsive adaptation

La página se adapta a dispositivos móviles mediante reglas CSS que ordenan el menú, las tarjetas y la tabla para mejorar la lectura en pantallas pequeñas.
